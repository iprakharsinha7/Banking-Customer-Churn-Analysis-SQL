# Customer Churn Analytics for Banking Industry

## Overview
This project analyzes customer churn behavior in the banking industry using PostgreSQL and SQL. The objective was to identify churn drivers, high-risk customer groups, and business-focused retention strategies.

---

## Tools & Technologies
- PostgreSQL
- SQL
- pgAdmin
- Window Functions
- Data Analysis

---

## Key Analysis Performed
- Customer churn analysis
- Summary statistics
- Customer segmentation
- Geography-based churn trends
- Active vs inactive customer analysis
- High-value customer identification using window functions

---

## Key Insights
- Churned customers held higher balances than retained customers.
- Inactive customers showed significantly higher churn tendencies.
- Germany exhibited higher churn concentration compared to other regions.
- High-value customers were also found among churned segments.

---

## Business Recommendations
- Build targeted retention strategies for high-balance customers.
- Create inactivity-trigger engagement systems.
- Develop region-specific retention campaigns.

---

## Outcome
This project demonstrates how SQL and PostgreSQL can be used to transform raw customer data into actionable business insights and strategic recommendations.
